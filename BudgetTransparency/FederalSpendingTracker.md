# Name

Federal Spending Tracker

# Purpose

Server application to serve U.S. federal spending data via a RESTful API. USAspending is the official open data source of federal spending information,
including information about federal awards such as contracts, grants, and loans.

# Geography

United States

# URL

www.usaspending.gov
